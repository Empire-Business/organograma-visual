# Design System V2.0

> Sistema de design profissional estilo Linear/Vercel
> Última atualização: 2026-02-20

---

## Visão Geral

O Design System V2.0 introduz um sistema multi-accent que permite cores diferentes por área de negócio, mantendo consistência visual em toda a aplicação.

---

## Cores por Área

### Áreas Principais

| Área | Cor Primary | Cor Light | Cor Dark | Uso |
|------|-------------|-----------|----------|-----|
| **Aquisição** | `#8B5CF6` (Violet) | `#EDE9FE` | `#6D28D9` | Vendas, marketing, captação |
| **Entrega** | `#10B981` (Emerald) | `#D1FAE5` | `#059669` | Projetos, execução |
| **Operação** | `#F59E0B` (Amber) | `#FEF3C7` | `#D97706` | Suporte, infraestrutura |
| **Geral** | `#6B7280` (Gray) | `#F3F4F6` | `#374151` | Elementos neutros |

### Paletas de Cores Accent

Além das cores de área, o sistema oferece múltiplas paletas:

- **Violet** - C default
- **Emerald** - Sucesso, natureza
- **Amber** - Atenção, warning
- **Rose** - Erro, crítica
- **Sky** - Informação
- **Slate** - Neutros, texto

---

## Elevation System

Sistema de sombras em camadas para profundidade visual:

| Level | Sombreamento | Uso |
|-------|--------------|-----|
| `level1` | `0 1px 2px 0...` | Cards, botões |
| `level2` | `0 4px 6px -1px...` | Hover, dropdowns |
| `level3` | `0 10px 15px -3px...` | Modais, popovers |
| `level4` | `0 20px 25px -5px...` | Sidebars, overlays |
| `level5` | `0 25px 50px -12px...` | Dialogs críticos |

---

## Motion Design

### Durações

| Token | Valor | Uso |
|-------|-------|-----|
| `instant` | 0ms | Sem transição |
| `fastest` | 75ms | Micro-interações |
| `fast` | 150ms | Hover, focus |
| `normal` | 200ms | Transições padrão |
| `slow` | 300ms | Paineis, drawers |
| `slowest` | 500ms | Animações complexas |

### Easing Curves

| Token | Curva | Uso |
|-------|-------|-----|
| `smooth` | `cubic-bezier(0.4, 0, 0.2, 1)` | Padrão |
| `accelerate` | `cubic-bezier(0.4, 0, 1, 1)` | Entrada |
| `decelerate` | `cubic-bezier(0, 0, 0.2, 1)` | Saída |
| `elastic` | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` | Efeitos bounce |
| `back` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | Overshoot leve |

## Componentes V2

### StatusDot

Indicador de status com animação de pulse opcional.

```tsx
import { StatusDot } from '@/components/ui/status-dot'

// Estados: online, offline, busy, away
<StatusDot status="online" size="md" showPulse />
```

### MetricBadge

Badge de métrica com ícone e tendência.

```tsx
import { MetricBadge } from '@/components/ui/metric-badge'

<MetricBadge
  label="Projetos"
  value={12}
  area="entrega"
  icon={<FolderIcon />}
  trend="up"
  trendValue="+3"
/>
```

### ProgressBar

Barra de progresso com animação shimmer.

```tsx
import { ProgressBar } from '@/components/ui/progress-bar'

<ProgressBar
  value={75}
  area="aquisicao"
  showLabel
  showValue
/>
```

### BentoCard

Card estilo Bento Grid com gradiente e shine effect.

```tsx
import { BentoCard, BentoGrid, BentoStat } from '@/components/ui/bento-card'

<BentoGrid cols={3} gap="md">
  <BentoStat
    title="Total Projetos"
    value={24}
    area="entrega"
    trend={{ value: '+12%', direction: 'up' }}
  />
</BentoGrid>
```

### PersonCardV2

Card de pessoa com badges de métricas e avatar ring.

```tsx
import { PersonCardV2 } from '@/components/pessoa/person-card-v2'

<PersonCardV2
  nome="João Silva"
  cargo="Desenvolvedor"
  area="entrega"
  status="online"
  projetos={5}
  projetosConcluidos={3}
  tarefas={12}
  tarefasConcluidas={8}
/>
```

## Como Usar

### 1. Importar tokens

```tsx
import { areaColors, elevation, motion } from '@/design/tokens-v2'

// Usar cor de área
const areaColor = areaColors.entrega

// Usar elevation
style={{ boxShadow: elevation.level2 }}
```

### 2. Usar componentes

```tsx
import { StatusDot, BentoCard } from '@/components/ui'

// Componente simples
<StatusDot status="online" />

// Componente com área
<BentoCard area="aquisicao">
  Conteúdo
</BentoCard>
```
